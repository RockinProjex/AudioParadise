# AudioParadise
An all new, modern reimplementation of audio software for DAPs, MP3 Players, Phones/Tablets and other miscellaneous digital music libraries

# Welcome
Hello! I am RockinProjex here with another repo (I have far to many...) and this time it is for a small hobby side project I call AudioParadise.

I myself do not know many details of what it'll be just yet, but I know it'll function like iTunes did during the height of the iPod Classic, and if you have my fork of Clementine (Fruity Player) installed and you point AudioParadise to its executable it'll start playing your selected music.

This project will be written in Python and TypeScript, and at first it will only have compatability for Windows 11. I will expand upon that later.

# Development
This project is in very early development. I can't remember the last time I used python or typescript for a massive project so I'll probably be a little slow developing it to start with.

# Features
- a sleek, modern UI design
- syncing with many devices (or even a folder)
- tag editing capability
- links to useful sources for tag data and album art
(this is not intended to be a replacement for MusicBrainz Picard, Fre:Ac, MP3tag or any other tagging software)
- customisable themes
- lots more...

That's all I have for now, check back later once the project is truely underway!
